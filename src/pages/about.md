I'm **Himanshu Kumar** — most people know me as **nycanshu** online. I'm a Software Engineer at [Tower](https://www.linkedin.com/in/okay-anshu/), working primarily on backend systems with Java (Spring Boot) and Node.js/TypeScript.

## What I do

I build backend services, SDKs, and infrastructure tooling. I'm particularly interested in distributed storage systems (Ceph), Kubernetes, and developer experience.

Some things I've shipped:

- [**radosgw-admin**](https://radosgw-admin.nycanshu.dev) — A zero-dependency Node.js SDK for the Ceph RADOS Gateway Admin API. Built with TypeScript, AWS Signature V4 auth, dual ESM/CJS. ([wrote about it on dev.to](https://dev.to/okay_anshu/i-built-a-typescript-client-for-ceph-object-storage-because-the-only-npm-package-was-7-years-old-1nh0))
- **componentverse** — Power Platform Components Library for Canvas app developers
- Various freelance projects spanning healthcare management, loan systems, and data processing tools
